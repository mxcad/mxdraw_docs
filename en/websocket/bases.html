    WebSocket (MxCAD) Editing Mode (Deprecated, for internal use only)

    # WebSocket (MxCAD) Editing Mode (Deprecated, for internal use only)

    Prerequisites

    • MxDraw Cloud Development Kit (opens new window) is required to start the WebSocket service.

    • Mx.MxFun.setIniset({ EnableIntelliSelect: true}) enables selecting graphic objects for editing.

    • Mx.MxFun.setMxServer("ws://localhost:5090") sets the address of the WebSocket backend service for communication.

    # Start the WebSocket backend service

    Double-click the MxServer.exe file in the Bin\Release folder of the MxDraw Cloud Development Kit (opens new window) (click to go to the download page (opens new window)) to start the service.

    The default port is 5090. You can find the contents of the mxConfig.js file in the Bin\Release directory as follows:

    function MxConfig() {
    // Whether it is in debug mode.
    this.isDebug    = true;
    // Load test service
    this.isLoadTest = true;
    // Load MxCAD backend service
    this.isLoadMxCAD = true;
    // Server port for drawing conversion
    this.convertServerPort = 2600;
    // Backend service port
    this.backstageServerPort = 2700;
    // This is the port number of node, and the port number of WebSocket is this.nodeServerPort - mxweb.getId() = 5090 by default.
    this.nodeServerPort = 6000;
    if (this.backstageServerPort > this.nodeServerPort)
        this.backstageServerPort = this.nodeServerPort - 1;

    this.init = function (mxweb) {
        // Initial file search path These directory stores one drawing file each (please jump to the corresponding directory to see)
        mxweb.addFileSearchPath('../../SRC/TsWeb/public/demo');
        mxweb.addFileSearchPath('../Test');
        // It is not recommended to put drawings on another computer in the LAN. If this computer cannot be accessed, the server program will have a timeout waiting for access.
        //mxweb.addFileSearchPath('\\\\192.168.2.5\\data');
    };
};

var mxConfig = new MxConfig();
module.exports = mxConfig;

    # Enable MxCAD editing mode

    Add the configuration parameter isMxCAD with a value of true when calling the Mx.MxFun.createMxObject method to enable the editing mode.

    Also, call Mx.MxFun.setMxServer("ws://localhost:5090") to set the address of the WebSocket backend service.

    Here's an example code:

    import Mx from "mxdraw"

Mx.loadCoreCode().then(()=> {
    // Set the address of the WebSocket backend service
    Mx.MxFun.setMxServer("ws://localhost:5090");
    Mx.MxFun.createMxObject({
        canvasId: "myCanvas",
        isMxCAD: true,
        callback(mxDraw, { canvas }) {
            // Listen to the callback event from the backend service
            mxDraw.addEvent("serverReturn", function (data) {
                if (data.type == "ServerProcessCreate") {
                    if (data.val.succeeded) {
                        // When the backend service process is created successfully, open the drawing.
                        mxDraw.getMxCAD().openFile("test.dwg");
                    }
                }
            });
        },
    });
})
